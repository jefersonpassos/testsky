## Api desenvolvida usando AdonisJS

### Iniciar api
 
1. Criar arquivo *.env*
  
  > Utilizar .env.exemple como base 

2. `npm i`
3. `npm start`

### Teste
 `npm test`

 ## Rotas

 | Url | Metodo | descricao |
 |--|--|--|
 | /users | post | Cria usuario |
 | /users/:id | get | consulta usuario |
 | /sessions | post | faz login e devolve o token |
